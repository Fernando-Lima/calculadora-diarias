# Bem vindo ao projeto Calc Diária

## Informações do projeto

**URL**: https://diaria-prefeitura.lovable.app/

Projeto desenvolvido inteiramente por inteligência artificial com o propósito de facilitar no cálculo de diárias de servidores públicos. Esse projeto é exclusivamente para setores públicos que necessitam calcular o valor de diárias.

There are several ways of editing your application.

**Políticas de uso**

O proprietário desse projeto (Eu) não tem responsabilidade alguma por erro de cálculos.

As informações não ficam armazenadas em banco de dados nem em storge local.

Não é necessário informar dados pessoais dos servidores nem mesmo dados pessoais de usuários que utilizarão a plataforma.

Há limite de uso do projeto de acordo com a disponibilidade da plataforma Lovable.app do plano Free.

Em caso de falhas e instabilidade o administrador do projeto deve ser contactado para avaliar o que pode ser feito para corrigir.
